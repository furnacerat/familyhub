# Family Hub

A private household web app for shared family organization, adult budgeting, kid-friendly money habits, and home maintenance reminders.

## Current Stack

- Next.js App Router
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- lucide-react icons

## Run Locally

```bash
npm run dev
```

Then open [http://127.0.0.1:3000](http://127.0.0.1:3000).

## Initial App Areas

- Family dashboard
- Shared calendar snapshot
- Shopping list
- Reminders
- Adult-only budget companion
- Kids money goals
- Home maintenance reminders
